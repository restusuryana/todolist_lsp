# Aplkasi To-Do-List

##Deskripsi
Aplkasi sederhana berbasis php untuk mencatat tugas harian

##Fitur
-Tambah tugas
-Tandai tugas selesai
-Hapus tugas

##Struktur folder
-'index.php' - halaman utama

##cara menjalankan tugas
1.Salin folder ke 'htdocs/'
2. Jalankan XAMPP dan buka di 'http://localhost/todolist'
